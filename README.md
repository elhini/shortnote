Lite and simple app for short notes

## Current features:

- cloud DB storage

#### Fields:
- title
- text
- create date
- update date
- tags

#### Functions:
- filtering
- sorting
- routing (link to note)

## Planned features:

- design (bootstrap/material-UI)

#### Fields:
- group/category (with hierarchy)

#### Functions:
- auth (by login/password)
- admin panel
- multiple accounts (users)
- invitations
- public access
